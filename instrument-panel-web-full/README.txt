# Instrument Panel Web

✅ Java Spring Boot + Thymeleaf
✅ SQLite DB for real-time virtual car dashboard
✅ Auto refresh every 5 seconds

## Launch
1. Open this folder as a Maven project via pom.xml
2. Make sure `instrument.db` is in root folder
3. Run `InstrumentPanelWebApplication.java` or `mvn spring-boot:run`
4. Open http://localhost:8080/dashboard
